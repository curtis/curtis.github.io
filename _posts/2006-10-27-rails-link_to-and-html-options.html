Just a small thing, but I needed to know how to do this and it took me a while to find. Hopefully this will help someone else out.

In HTML, we might declare a link like so

{% highlight html %} Click Here! {% endhighlight %}

Now, in Ruby on Rails, we want to declare a link, so we use the built in link_to method like this

{% highlight erb %} <%= link_to "Click Here!", :controller => "controller", :action => "action" %> {% endhighlight %}

Hmmmâ€¦ how do we squeeze a title in there? Luckily, the third parameter to link_to is a hash of HTML options.

{% highlight erb %} <%= link_to "Click Here!", {:controller => "controller", :action => "action"}, {:title => "Example"} %> {% endhighlight %}
